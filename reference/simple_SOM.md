# Example objects from salmonMSE

S4 objects containing operating model inputs to run the projection.

## Usage

``` r
simple_SOM

multi_SOM
```

## Format

An object of class
[SOM](https://docs.salmonmse.com/reference/SOM-class.md).

An object of class `SOM` of length 1.
